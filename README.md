# Claude Wrapped 2025

See your year in conversation with Claude AI. A Spotify Wrapped-style experience for your Claude chat history.

## Features

- **Stats Overview**: Total chats, messages sent, chattiest day, days active
- **Activity Heatmap**: GitHub-style visualization of your chat activity
- **Archetype**: Discover your conversation style (Sage, Paladin, Tactician, Ranger, etc.)
- **Top Themes**: See what topics you discussed most
- **Iconic Phrases**: Your most-used words and phrases
- **Awards**: Fun awards based on your usage patterns
- **AI Poem & Predictions**: Personalized poem and 2026 predictions generated by AI

## Quick Start

1. Export your conversations from Claude (Settings > Export Data)
2. Open `index.html` in your browser (or run a local server)
3. Drop your `conversations.json` file
4. Navigate through your wrapped experience!

```bash
cd claude-wrapped
python3 -m http.server 8000
# Open http://localhost:8000
```

## AI Features (100% Local)

The poem and predictions are generated using **SmolLM2-135M-Instruct**, a small language model that runs entirely in your browser.

- **No API key required** - everything runs locally
- **~150MB download** on first use (cached for future visits)
- **WebGPU accelerated** on modern browsers
- **Falls back to WASM** on older browsers

### Browser Support

| Browser | Support |
|---------|---------|
| Chrome 113+ | WebGPU (fastest) |
| Edge 113+ | WebGPU |
| Firefox | WASM (slower) |
| Safari 18+ | WebGPU |
| Mobile Chrome | WebGPU on newer devices |
| Mobile Safari | WASM fallback |

If your browser doesn't support local AI, template-based content will be used instead.

## Privacy

**100% Private**: Your conversation data never leaves your browser. All processing happens locally:
- Data extraction: JavaScript in your browser
- AI generation: Local model via [Transformers.js](https://huggingface.co/docs/transformers.js)

No data is sent to any server.

## Deployment

### Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd claude-wrapped
vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.

### Other Static Hosts

Works on any static host (Netlify, GitHub Pages, etc.) - just upload the files.

## Tech Stack

- Vanilla JavaScript (no build step)
- [Transformers.js](https://huggingface.co/docs/transformers.js) - Local LLM inference
- [SmolLM2-135M-Instruct](https://huggingface.co/HuggingFaceTB/SmolLM2-135M-Instruct) - AI model
- CSS animations and gradients
